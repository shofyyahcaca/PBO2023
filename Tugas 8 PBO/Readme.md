#Persistence
Java Persistence API (JPA) membantu dalam mengelola data relasi dengan menggunakan POJO (Plain Old Java Object) dengan menyediakan fasilitas untuk melakukan object-relational mapping (ORM) yang memungkinkan pengembang Java untuk mengakses data dari database menggunakan objek Java. Dengan menggunakan JPA, pengembang dapat melakukan mapping antara objek Java dengan tabel di database relasional. JPA memudahkan pengembang untuk melakukan operasi CRUD (Create, Read, Update, Delete) pada data dengan mudah dan efisien.
